# ThreeJsGame

This is a simple 3D browser game built with Vanilla Three.js for the i2i Systems Academy assignment. 

In this game, you control a pink cube on a restricted ground area and try to dodge the brown obstacle cubes coming at you. The longer you survive, the higher your score!

## Features
* **Vanilla Three.js:** Built with standard JavaScript and Three.js, without extra frameworks.
* **Movement Boundaries:** The player is restricted to the safe play area and cannot fall off the edges.
* **Collision Detection:** The game detects when you hit a brown box and shows a Game Over screen.
* **Score Tracking:** Your score automatically increases as you stay alive.

## Controls
* **W, A, S, D** or **Arrow Keys**: Move the pink cube.
* **Spacebar**: Restart the game after losing.

## How to Run the Project locally
1. Clone this repository to your computer.
2. Open your terminal inside the project folder.
3. Run `npm install` to download the required packages.
4. Run `npm run dev` to start the local server.
5. Open the localhost link in your browser and play!
